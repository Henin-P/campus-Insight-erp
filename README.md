# ЁЯОУ Student ERP Portal

A comprehensive Student ERP (Enterprise Resource Planning) system developed using **Python, Streamlit, SQLite, Pandas, and Plotly**. The application provides an integrated platform for managing student records, attendance, academic performance, reporting, and data analytics through an interactive web-based dashboard.

---

## ЁЯЪА Project Overview

The Student ERP Portal is designed to simplify academic record management by providing a centralized system for:

* Student Information Management
* Attendance Tracking
* Marks Management
* Academic Performance Analysis
* Data Visualization & Reporting
* Bulk Student Data Import via Excel

The project demonstrates practical implementation of database management, data analytics, reporting systems, and dashboard development using Python.
